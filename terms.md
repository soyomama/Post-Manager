# 利用規約（Terms of Service）

本サービス（以下「本アプリ」）は、ユーザーが自身の動画コンテンツを管理し、投稿前に内容を確認した上でTikTokへ共有するためのツールです。本アプリを利用することで、ユーザーは以下の内容に同意したものとみなします。

## 1. サービス内容
本アプリは、ユーザーが選択した動画・キャプション・公開設定を確認し、ユーザー自身が投稿ボタンを押した場合にのみ、TikTok Content Posting API を使用して動画を共有します。

## 2. 自動投稿は行いません
本アプリは、ユーザーの明示的な操作なしにTikTokへ投稿を行うことはありません。

## 3. TikTok API の利用
本アプリは TikTok Developer Platform を使用しており、利用にあたり TikTok の利用規約およびポリシーに従います。

## 4. 禁止事項
- 不正アクセスや不正利用  
- TikTok のコミュニティガイドラインに反する投稿  
- 他人の著作物を無断でアップロードする行為

## 5. 免責事項
本アプリの利用により発生した損害について、提供者は一切の責任を負いません。
