# プライバシーポリシー（Privacy Policy）

本アプリは、ユーザーの動画投稿を支援するために、以下の情報を取り扱います。

## 1. 取得する情報
- TikTok OAuth により取得される open_id、access_token、refresh_token  
- アプリ上でユーザーが入力する動画・キャプション・公開設定  

## 2. 利用目的
取得情報は、ユーザーが投稿ボタンを押した場合に限り TikTok へ投稿するために使用します。

## 3. データの保存と保護
アクセストークンは安全に保存され、投稿目的以外に利用しません。

## 4. 第三者提供
法律で求められる場合を除き、外部に提供することはありません。

## 5. データ削除
ユーザーは問い合わせによりデータ削除を依頼できます。

## 6. TikTok API の利用
本アプリは TikTok Developer Platform を利用し、TikTok 規約に準拠します。
